# 快捷键指南 (Shortcuts Guide)

ClipFlow 提供了一系列快捷键来提高您的工作效率。您可以在 `设置` -> `常规` -> `全局快捷键` 中自定义部分快捷键。

## 全局快捷键 (Global Shortcuts)

这些快捷键可以在任何应用程序中使用，只要 ClipFlow 正在运行。

| 功能 | 默认快捷键 (macOS) | 说明 |
| --- | --- | --- |
| **显示/隐藏窗口** | `Cmd + Alt + \`` | 快速唤起或隐藏 ClipFlow 主窗口（`键通常在 Tab 键上方） |
| **快速粘贴** | `Cmd + Ctrl + V` | 粘贴剪贴板历史中最近的一项内容 |
| **显示历史记录** | `Cmd + F9` | 打开完整的剪贴板历史记录界面 |
| **搜索内容** | `Cmd + Shift + F` | 直接打开搜索栏，查找历史记录 |
| **OCR 文字识别** | `Cmd + F8` | 对当前剪贴板中的图片进行文字识别 |

## 应用内快捷键 (In-App Shortcuts)

当 ClipFlow 窗口处于活动状态（前台）时可用的快捷键。

| 功能 | 快捷键 | 说明 |
| --- | --- | --- |
| **隐藏窗口** | `Cmd + H` | 隐藏应用窗口（系统标准快捷键） |
| **退出应用** | `Cmd + Q` | 完全退出 ClipFlow |
| **关闭窗口** | `Cmd + W` | 关闭当前窗口（通常最小化到托盘） |
| **打开设置** | `Cmd + ,` | 打开设置页面 |
| **全选** | `Cmd + A` | 全选文本（如在搜索框中） |
| **复制** | `Cmd + C` | 复制选中的内容 |
| **粘贴** | `Cmd + V` | 粘贴内容 |
| **剪切** | `Cmd + X` | 剪切选中的内容 |
| **撤销** | `Cmd + Z` | 撤销上一步操作 |

## 快捷键设置与冲突

### 自定义快捷键
您可以在设置中修改全局快捷键。点击对应的快捷键设置项，然后按下您想要设置的新组合键即可。

### 避免冲突
为了保证系统稳定性，ClipFlow 禁止注册以下关键系统快捷键作为全局快捷键：
*   `Cmd + C` (复制)
*   `Cmd + V` (粘贴)
*   `Cmd + X` (剪切)
*   `Cmd + Z` (撤销)
*   `Cmd + A` (全选)
*   `Cmd + S` (保存)
*   `Cmd + Q` (退出)
*   `Cmd + Tab` (切换应用)

### 常见问题
*   **快捷键不工作**：请检查是否有其他应用占用了相同的快捷键，或者尝试在设置中重新设置。
*   **Cmd + H 隐藏后如何找回**：
    *   使用全局快捷键 `Cmd + Alt + \``
    *   点击菜单栏图标
    *   点击 Dock 栏图标
