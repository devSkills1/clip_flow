### `ClipboardService` 优化
1. 自适应轮询策略

- 实现动态轮询间隔调整（250ms-2000ms）
- 空闲时自动降低检查频率，活跃时提高响应速度
- 减少不必要的系统资源消耗
2. 图片比较算法优化

- 使用 SHA-256 哈希值替代逐字节比较
- 小图片（<10KB）仍使用字节比较保证精度
- 大幅提升大图片处理性能
3. 剪贴板变化检测

- 添加序列号和时间戳检测机制
- 快速判断剪贴板是否发生变化
- 避免不必要的内容获取操作
4. 内容缓存机制

- 实现基于哈希的内容缓存（最大100项，5分钟过期）
- 避免重复处理相同内容
- 显著提升响应速度
5. 异步处理优化

- 使用 Isolate 处理耗时的哈希计算
- 避免阻塞主线程，保持UI流畅
- 提升整体用户体验
